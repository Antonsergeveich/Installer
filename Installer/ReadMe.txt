Эта программа позволяет установить два приложения:

1. 7-Zip - архиватор файлов;
2. XnView MP - просмотрщик и конвертер изображений;

Программа проверяет, установлены приложения или нет, и пропускает их при повторном запуске.
Все устанощики скачиваются в папку "Загрузки".
После установки временные файлы удаляются.

Инсталлятор может работать в двух режимах:

Обычный режим с интерфейсом;
Тихий режим без интерфейса для автоматической установки всех приложений;

Для того чтобы воспользоваться тихим режимом запустите программу через командную строку
с параметром /S или /verysilent;
Пример:

Installer.exe /S
Installer.exe /verysilent

Разницы нет, работают одинаково.
Незабудьте указать полный путь до файла!

Программа поддерживает все версии Windows, начиная с Windows 7 (x86/x64).

Для работы установщика убедитесь, что Ваш компьютер подключён к интернету,
так как программа автоматически загружает актуальные версии приложений с официальных источников.
